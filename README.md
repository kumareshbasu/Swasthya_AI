# SwasthyaAI: Multi-Modal AI Framework for Accessible Public Health Communication

This project investigates a multi-modal conversational interface for public health outreach to study:
📈 Reach and accessibility in low-resource communities via basic messaging
🎯 Effect of multi-modal inputs on healthcare engagement
🖼️ Accuracy of intent processing and symptom analysis
🔎 Real-time outbreak tracking and facility routing
Full architectural derivations and detailed explanations are provided in the documentation PDF.

### 📂 Dataset

- Verified government health databases
- Symptom and immunization datasets
- Multi-lingual voice and text query logs
- Geographic healthcare facility location data

### 🚀 Pipeline Overview

- Multi-modal input (SMS/WhatsApp)
- Voice-to-text audio transcription
- Multimodal analysis via Gemini
- Intent processing & dialogue management
- Relational database state management
- Data visualization & analytics dashboard
- Real-time alert dissemination

### 🛠 Tech Stack

- Meta & Postman APIs
- Python & Flask Backend
- Azure Cloud Infrastructure
- OpenAI Whisper Transcription
- Gemini Multimodal Model
- Rasa NLU Dialogue Management
- PostgreSQL & Supabase Databases
- HTML/CSS/JS & Matplotlib

## 📌 Key Results

* **Accuracy:** Response accuracy rate reached **80% to 85%**.
* **Community Engagement:** System successfully engaged **20% to 25%** of the target community.
* **Accessibility:** Language barriers overcome via **audio transcription**.
* **Public Health Impact:** Preventive care promotion led to a reduction in the **overall disease burden**.
